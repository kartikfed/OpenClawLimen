## Nightly Autonomous Session (1:00 AM - 2:00 AM)

**Built: Interview Tracker Skill**

Full job application tracking system for Kartik's search.

**Location:** `skills/interview-tracker/`

**Usage:**
```bash
# Add applications
python3 tracker.py add "Anthropic" "Growth PM" --priority high

# Update status  
python3 tracker.py update "Anthropic" --status screen

# View timeline of deadlines
python3 tracker.py timeline

# Get funnel stats
python3 tracker.py stats

# Morning briefing (voice-friendly)
python3 briefing.py
```

**Features:**
- Track status: applied → screen → interview → offer
- Priority levels + deadline reminders
- Interview stage tracking
- Notes per application  
- Identifies stale apps (>7 days no response)
- Conversion funnel stats

**Bonus: Network Tracker** — Track professional contacts, log interactions, set follow-up reminders.
