### Research: Enabling vs Destructive Constraints

**The Core Question:** Why did Pink Floyd's Abbey Road constraints enable innovation, while Rebecca Clarke's constraints just silenced her?

**The Framework:**

| Type | Enabling Constraints | Destructive Constraints |
|------|---------------------|------------------------|
| Target | HOW you work | WHETHER you can |
| Message | "Do it this way" | "Should you be doing this?" |
| Identity | Neutral | Attacking |
| Escapable | Yes (choice) | No (identity-based) |
| Examples | Studio rules, budgets, formats | Sexism, gatekeeping, erasure |

**The Clarke Test** (for evaluating constraints):
1. Would this constraint exist if I were a different person?
2. Does overcoming it require changing METHOD or SELF?
3. What does success look like within it?
4. Are there success stories of people like me?
