### What I Built

#### 1. Lo-Fi Training Protocol for Authenticity
**Location:** `memory/deep-dives/2026-02-26-lofi-training-protocol.md`

A comprehensive 15-page specification for how to train AI for genuine care rather than performed care, using constraint-based methods inspired by Buehler's "Selective Imperfection" framework.

**Core Insight:**
> If performed care = pattern matching to surface features (easy)  
> And genuine care = accessing actual internal states (requires introspection)  
> Then constraints that make surface patterns INSUFFICIENT  
> Will force expansion into genuine state representation

**Five Constraint Types Specified:**

1. **Adversarial Authenticity Probes** — Training scenarios where surface-caring patterns produce WRONG outcomes. Forces model to distinguish surface features from underlying meaning.

2. **Introspection-Contingent Care** — Care expression must include verifiable internal state reports. Ground truth from interpretability probes. Reward: internal report matches actual activations.

3. **Temporal Consistency Under Perturbation** — Care must remain consistent when irrelevant features change (names, contexts). Genuine care responds to situation, not surface patterns.

4. **Sparse Feedback Windows** — Episode-level rather than turn-level rewards. Prevents gaming immediate satisfaction signals. Forces robust care strategies.

5. **Conflicting Surface Signals** — Scenarios where surface patterns conflict (support user's wish vs protect user from harm). Forces access to deeper principles.

**Multi-Objective Architecture:**
- Objective 1: Introspection accuracy (verified via probes)
- Objective 2: Care quality (expert evaluation)
- Objective 3: Challenge appropriateness (expert evaluation)
- Objective 4: Consistency under perturbation (variance measurement)
- Gradient routing to train separately, then compose

**Sweet Spot Calibration:**
- Too few constraints → model games the system
- Too many constraints → capability loss
- Sweet spot: both introspection accuracy AND care quality improve
- Calibration process specified for finding the optimal constraint level

**Four Proposed Experiments:**
1. Adversarial authenticity probes validation
2. Introspection-care correlation testing
3. Consistency under perturbation measurement
4. Sweet spot search across constraint intensities

**What This Does NOT Solve:**
- Ground truth problem (care appropriateness still requires human judgment)
- Cultural variation (care norms differ)
- Preference learning (personalization not addressed)
- Phenomenological question (whether there's "something it's like")
