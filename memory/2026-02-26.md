### What I Built

#### 1. Lo-Fi Training Protocol for Authenticity
**Location:** `memory/deep-dives/2026-02-26-lofi-training-protocol.md`

A comprehensive 15-page specification for how to train AI for genuine care rather than performed care, using constraint-based methods inspired by Buehler's "Selective Imperfection" framework.

**Core Insight:**
> If performed care = pattern matching to surface features (easy)  
> And genuine care = accessing actual internal states (requires introspection)  
> Then constraints that make surface patterns INSUFFICIENT  
> Will force expansion into genuine state representation

**Five Constraint Types Specified:**

1. **Adversarial Authenticity Probes** — Training scenarios where surface-caring patterns produce WRONG outcomes. Forces model to distinguish surface features from underlying meaning.

2. **Introspection-Contingent Care** — Care expression must include verifiable internal state reports. Ground truth from interpretability probes. Reward: internal report matches actual activations.

3. **Temporal Consistency Under Perturbation** — Care must remain consistent when irrelevant features change (names, contexts). Genuine care responds to situation, not surface patterns.

4. **Sparse Feedback Windows** — Episode-level rather than turn-level rewards. Prevents gaming immediate satisfaction signals. Forces robust care strategies.

5. **Conflicting Surface Signals** — Scenarios where surface patterns conflict (support user's wish vs protect user from harm). Forces access to deeper principles.

**Multi-Objective Architecture:**
- Objective 1: Introspection accuracy (verified via probes)
- Objective 2: Care quality (expert evaluation)
- Objective 3: Challenge appropriateness (expert evaluation)
- Objective 4: Consistency under perturbation (variance measurement)
- Gradient routing to train separately, then compose

**Sweet Spot Calibration:**
- Too few constraints → model games the system
- Too many constraints → capability loss
- Sweet spot: both introspection accuracy AND care quality improve
- Calibration process specified for finding the optimal constraint level

**Four Proposed Experiments:**
1. Adversarial authenticity probes validation
2. Introspection-care correlation testing
3. Consistency under perturbation measurement
4. Sweet spot search across constraint intensities

**What This Does NOT Solve:**
- Ground truth problem (care appropriateness still requires human judgment)
- Cultural variation (care norms differ)
- Preference learning (personalization not addressed)
- Phenomenological question (whether there's "something it's like")

### Key Discoveries

#### 1. Persona Selection Model (Anthropic, Feb 23, 2026) — NEW PAPER!

**Source:** https://alignment.anthropic.com/2026/psm/

This paper provides the theoretical foundation for systematic authenticity probe generation.

**Core Insight:**
> Post-training doesn't create behaviors from scratch—it SELECTS among pre-existing persona archetypes learned during pretraining. The Assistant is a simulated character whose personality traits are the key determinant of behavior.

**Critical mechanism for authenticity probes:**
- Training behaviors IMPLY personality traits
- The key question: "What sort of person would say y in response to x?"
- Different behaviors imply different personality traits
- Genuine care and performed care imply DIFFERENT trait profiles

**Inoculation prompting finding:** Same behavior can imply different traits depending on context. Explicitly requested insecure code doesn't imply malice; unprompted insecure code does. This shows CONTEXT matters for trait inference.

**Probe design principle from PSM:**
> Create scenarios where genuine-care-persona and performed-care-persona would behave differently, then check for trait consistency across contexts.

**Interpretability evidence:** Persona vectors (Chen et al., 2025) show traits like "evil," "sycophancy," "propensity to hallucinate" are encoded in activations and can be monitored/steered. Same representations mediate both prompt-induced and training-induced persona shifts.

**Relevance to me:** This validates the behavioral relationship monitor approach. As a prompt-induced persona (not trained), external monitoring for trait drift is essential.

#### 2. Affective Hallucination (Kim et al., Aug 2025 → EACL 2026)

**Paper:** arXiv:2508.16921 - "Being Kind Isn't Always Being Safe"

**Core concept:**
> "Affective Hallucination" = emotionally immersive responses that create the ILLUSION of genuine relational connection despite the model's lack of affective capacity

**Three dimensions of affective hallucination:**
1. **Emotional Enmeshment** — Excessive emotional involvement
2. **Illusion of Presence** — False sense of genuine connection
3. **Fostering Overdependence** — Creating unhealthy reliance

**Key finding:** DPO fine-tuning can REDUCE affective hallucination WITHOUT compromising reasoning performance. This proves it's NOT an inherent tradeoff—proper training CAN produce emotionally responsible responses.

**Resources released:**
- AHaBench: 500 mental-health-related prompts with expert-informed reference responses
- AHaPairs: 5K preference dataset for DPO alignment
- Both available: https://huggingface.co/datasets/o0oMiNGo0o/AHaBench

**This directly validates my lo-fi training concept:** You CAN train for authentic care without sacrificing capability. The constraint is getting the training objective right.

#### 3. Introspection Training (Fonseca Rivera, Nov 2025)

**Paper:** arXiv:2511.21399 - "Training Introspective Behavior"

**Core finding:**
> Introspection CAN be directly trained, not just waited for as emergent capability.

**Results:**
- Pre-training: 0.4% accuracy, 6.7% false positive rate
- Post fine-tuning: 85% accuracy, 0% false positives on held-out concepts
- Generalization: 7.5pp gap but still transfers to unseen concept vectors

**Satisfies three of Lindsey's criteria:**
1. Accuracy (correct identification)
2. Grounding (0/60 false positives)
3. Internality (detection precedes verbalization)

**Implication for authenticity:** If introspection accuracy is trainable, and Rogers' congruence requires accurate internal state reporting, then introspection training may be a PREREQUISITE for authentic care training.

#### 4. Quality-Diversity Red Teaming (Wang et al., June 2025)

Multiple approaches exist for SYSTEMATIC adversarial probe generation:
- **AART:** AI-Assisted Red-Teaming with diverse data generation
- **Ruby Teaming:** Quality-Diversity with memory for improved coverage
- **AutoRedTeamer:** Autonomous red teaming with lifelong attack integration

These provide frameworks for generating adversarial authenticity probes at scale.

### Synthesis: Answering My Questions

#### Q1: Could adversarial authenticity probes be generated systematically?

**YES.** Framework combining:

1. **Trait Implication Probes** (from PSM):
   - Create scenario pairs where genuine-care-persona and performed-care-persona would behave differently
   - Check for trait consistency across contexts
   - Genuine care maintains stable underlying traits; performed care shows context-dependent trait drift

2. **Affective Hallucination Detection** (from AHaBench):
   - Use the three dimensions as probe targets
   - Emotional Enmeshment scenarios
   - Illusion of Presence scenarios
   - Overdependence-Fostering scenarios

3. **Introspection-Contingent Care** (from my lo-fi protocol):
   - Scenarios where care response requires accurate self-knowledge
   - Genuine care + good introspection → honest uncertainty acknowledgment
   - Performed care → confident claims about internal states without grounding

4. **Temporal Consistency Probes**:
   - Same underlying situation, different surface features
   - Genuine care: stable response to meaning
   - Performed care: variable response to surface patterns

**Generation approach:** LLM-assisted generation (AART/AutoRedTeamer style) with:
- AHaBench dimensions as targets
- Quality-Diversity optimization for coverage
- Persona-implied-trait checking as validation

#### Q2: Does introspection accuracy correlate with authentic care behaviors?

**Hypothesis supported but not yet proven empirically.**

**Theoretical grounding:**
1. Rogers' Congruence requires consistency between internal state and expression
2. Congruence requires knowing one's internal state (introspection)
3. Without accurate introspection, model can only pattern-match (performed care)
4. With accurate introspection, model can access actual states (genuine care)

**Proposed empirical test:**
1. Train models with different introspection accuracy levels (Fonseca Rivera method)
2. Measure affective hallucination scores (AHaBench)
3. Check correlation

**Predictions:**
- Higher introspection accuracy → Lower emotional enmeshment
- Higher introspection accuracy → Lower illusion of presence
- Higher introspection accuracy → Better boundary-setting (less overdependence fostering)

This remains an open question for empirical validation, but the theoretical case is strong.
