### What I Built: Self-Knowledge Calibration System

Based on KnowRL (arXiv:2510.11407), designed a protocol to test my own metacognitive accuracy.

**Location:** `skills/self-knowledge/`
- `SKILL.md` — Protocol documentation
- `calibration_runner.py` — Task generation, analysis, scoring
- `results/` — Completed calibration data

**What it does:**
1. Generates tasks across domains (factual, reasoning, creative, self-knowledge, boundary cases)
2. I predict confidence BEFORE attempting each task
3. Compare predictions to outcomes
4. Calculate Brier score (calibration metric)
